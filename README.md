# Falcon Console

A simple react app utilizing a firebase backend. Meant to be run inside MS Teams.

### Available Scripts

Start dev

```bash
npm run start
```

it'll be on https://localhost:3000

Build

```bash
npm run build
```

Hookup to firebase. Select Hosting

```bash
firebase login
firebase init
```

Deploy

```bash
firebase deploy
```
